## p5.rbで作った作品置場

https://ksbmyk.github.io/sketch/

詳細画面でPCからの場合のみ表示される🔄を押すとp5.rbでの描画になります。再度🔄を押すとp5.rbを再実行します
<img src="public/sketch.png">

## ローカル環境

- サーバー立ち上げ
  ```
  npm run dev
  ```
- 静的ページエクスポート
  ```
  npm run build
  ```
